Software RequiredGo Top

For edit this template I recommend you to use programs from Adobe.com. Also you can use any other existing programs. For example to edit html/css code you can use notepad++.

  • Adobe Dreamweaver CS+
  • Adobe Photoshop CS+
  • To uncompress ZIP package: WinZip 9 (Windows)

FeaturesGo Top

The main template features are:

  • All major devices are supported (e.g.: iMac, MacBook Pro, iPad, iPhone);
  • Easily customizable
  • Touch optimized & gesture support
  • HTML5 & CSS3
  • Absolutely free
  • PSD files included
  • Major screen resolution support (e.g.: Desktop layout 960px, Table layout 768px, Smartphone landscape layout 480px, Smartphone portrait layout 320px)
  • Browsers compatible (e.g.: Google Chrome, Mozilla Firefox, Safari, Opera, Internet Explorer)
  • Can be used like Facebook page
  • All templates was successfully checked as HTML5 and CSS level 3 Markup Validation Service on http://validator.w3.org
